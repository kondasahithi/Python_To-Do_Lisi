# To-Do List Application

A simple console-based **To-Do List Application** built using Python.  
This project allows users to add, view, and delete tasks using lists and loops.

## 📌 Features

- Add new tasks
- View all tasks
- Delete tasks
- Simple menu-driven interface
- Beginner-friendly project

## 🛠️ Technologies Used

- Python 3

## 📂 Project Structure

```text
todo_list.py
README.md
```

## ▶️ How to Run the Project

1. Install Python on your system
2. Save the file as `todo_list.py`
3. Open terminal or command prompt
4. Run the following command

```bash
python todo_list.py
```

## 💻 Sample Output

```text
===== TO-DO LIST =====
1. Add Task
2. View Tasks
3. Delete Task
4. Exit
```
